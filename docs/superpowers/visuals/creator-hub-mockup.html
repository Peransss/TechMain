Ringkasan CRUD & Multimedia:

  • Multimedia: Gambar diunggah ke Firebase Storage; ditampilkan di layar kuis.
  • Create: Tombol "＋" di tab Studio untuk membuat paket kuis baru.

  • Update: Tombol "✎" untuk mengubah judul, pertanyaan, atau gambar.
  • Delete: Fitur hapus (long-press) untuk mengelola penyimpanan cloud.
